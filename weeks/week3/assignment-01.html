Assignment 1: Wikipedia Clone

Brief: For our first lab project, you'll use the Wikipedia page from your class profile and redesign it.

Using the HTML and CSS we've learned so far, take the content from your wikipedia article and reimagine the layout and display.

Wikipedia is a massive and ever-expanding resource that has to contain so many different types of content. The design aims to be as neutral and flexible as possible to encapsulate all that content. How would you redesign your specific Wikipedia page to be more interesting, distinct, or opinionated?

Your redesign must have and will be graded on the following:

  • In a new html file in your github repository
  • Styles applied via an externally linked stylesheet
  • At least two linked fonts of your choosing
  • Preserve original content, but redesign it
  • A challenging and distinctive redesign. How well did you push your design past what Wikipedia did?
  • Properly composed HTML e.g. headings in h1-h6 tags, paragraphs in p tags, image captions using figure tags etc.. if you have a question about what element to use, refer to the MDN HTML Reference or inspect element with the dev tools to see what tag Wikipedia uses.

Things to consider:

  • Image size, layout
  • Citation links (anchor links to the bottom, display on hover, in the margins?)
  • Colors, margins, text alignment, table of contents...
  • How to style and structure tables, lists, charts...
  • CSS interactivity
  • Use inspect element from the developer tools to see how things are laid out
